# Atividade_JPA

Construa um pequeno sistema para uma concessionária de veículos utilizando JPA.

1. Concessionaria
    - Possui um endereco
    - Nome 
    - Telefone
    - Muitos veículos

2. Endereço
    - Rua
    - Numero
    - Bairro
    - Cidade
    - UF

3. Veículo
    - Marca
    - Modelo
    - Ano de fabricação
    - Muitos acessórios

4. Acessório
    - GPS
    - Ar-Condicionado
    - Vidro Eletrico
